# MERN E-Commerce App

## Features
- User authentication (JWT + cookies)
- Product management (admin)
- Cart system
- Order system
- Order status lifecycle

## Tech Stack
- Node.js
- Express
- MongoDB
- Mongoose
- React (frontend coming soon)

## Setup
1. Clone repo
2. Add `.env` file in backend
3. Run `npm install`
4. Start server
